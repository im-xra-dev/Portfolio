Evolve. [Open Source]
Evolve is a small project that I started working on when my internet was down. It aims to be a simulation of a planet containing animals, plants and bacteria. These organisms start with random stats, and over time will slowly evolve by breeding with other organisms to (hopefully) create a sustainable ecosystem that can be watched and observed. It is written in Java and is open-sourced at the link below.

Small World. [Open Source]
This is a small game that I created in 2 days for the Ludum Dare Compo on 22/4/17. This was the 38th compo which also marked 15 years since Ludum Dare started. The theme was to create something about a 'small world', so I chose to go with a tiny planet of which you are a character stuck on an island in a zombie outbreak. You must collect food to keep your family from dying for long enough to be rescued. The graphics are of low quality, and I could never get the food to appear in the exact right locations, however I am proud of it still as it was my first (and currently only) Ludum Dare game.

Password Generator.
For my GCSEs we were required to complete an NEA task, in which we had 20 hours to plan, write, and document a program that followed specific rules. This all had to be done in intense detail, and although the program itself was only a few hundred lines (I believe it was about 400) my documentation and plans were about 15 pages long. As you can probably tell, the program we had to write needed to be a password generator that could either generate a secure password (following given rules) or test the strength of a password (following the same rules). I managed to complete this task in about half the total time that we had, and got an 8 (2 marks off of a 9) overall in computing (Which is roughly an A*). As stated above, our school used Python for everything, so despite me asking to use Java, I wrote it in Python. Unfortunately, I do not have access to the program I wrote anymore, Otherwise it would be open-sourced.
